# Customer Behavior Analysis

## Problem Statement
Analyze customer shopping behavior to understand purchasing patterns, spending habits, and key factors influencing customer purchases.

## Dataset
Customer Shopping Behavior Dataset  
(Source: Public retail customer transaction data)

## Tools & Technologies
- Python (Pandas, NumPy)
- Exploratory Data Analysis (EDA)

## Key Business Questions
- Which customer segments spend the most?
- How does purchase amount vary by product category?
- Are there spending patterns based on age or gender?
- What factors appear to influence higher purchase amounts?

## Project Structure
- data/raw/ → original dataset
- notebooks/ → data loading, cleaning, and EDA notebooks
